2) Installation

Return to top

Extract the zipped package downloaded from ThemeForest to your desktop. In the extracted package you will find willow.zip which is the WordPress theme file.

You can install the theme one of two ways:

  • FTP Extract the willow.zip file and upload the extracted folder to the /wp-content/themes/ folder on your server.

  • WordPress Navigate to Apperance > Add New Themes > Upload page. Select the willow.zip file.
    Press the Install Now button to upload and install the theme.

After uploading the theme, you must activate it. Navigate to the Appearence > Themes page to activate the theme.

TIP: Reference this Themeforest page on how to correctly install a WordPress theme: ThemeForest Knowledgebase

3) Demo Content

Return to top

You can easily recreate the Willow demo site by importing the demo XML file. You can find the XML file in the zip folder you downloaded from Themeforest. After you've located the XML file, here's how you import it:

  • Step 1 Before you start make sure you have installed and activated the following plugins: WPBakery Visual Composer, Vafpress Portfolio, Contact Form 7.
  • Step 2 Go to Tools > Import > Click "WordPress". If you haven't already installed the WordPress import tool, you will be asked to install it.
  • Step 3 Once it's installed, activate and run the plugin. Now you will see a upload box, find and upload the willow-demo.xml file.
  • Step 4 Next you need to pick which author the demo content should be associated to and more importantly make sure to check the "Download and import file attachments" option.
  • Step 5 Now click submit. Please not that this can sometimes take a few minutes to install.

Now that the demo content has been installed, you're ready to set it up!

  • Setup the new homepage Go to Settings > Reading > Set "Front Page Displays" to A stactic Post > Choose the page called One Page which we just imported from the demo XML file.
  • Setup the menu Go to Appearance > Menus > Select the menu called Main > Assign the menu to the Header Navigation theme location > Also make sure you add the Section ID for each menu item (about, services, work, the-team, contact)
  • Setup the blog Go to Pages > Add New > Enter a title, like "Blog" > Click Publish > Go to Settings > Reading > Select "Blog" (or whatever you called your page) in the Posts Page dropdown.

4) Setup your one-page

Return to top

You can set up your One-page with these easy steps:

  • Step 1 Go to Pages > Add New > Enter a title, fx. "Home" > Choose "One Page" in the template dropdown option.
  • Step 2 Go to Settings > Reading > Set "Front Page Displays" to A stactic Post > Choose the page you just created.
  • Step 3 Now your newly created One-page is the frontpage of your site. You can now go back to edit the page and use the page builder to create your page.

In the next section called "Menus" we will cover how to create the auto scrolling navigation for your one-page.

5) Menus

Return to top

Willow supports one locations for custom WordPress menu - the Header Navigation:

For menu setup, navigate to the Appearance > Menus page. Enter a name for your menu and add links to the menu by using the options given on the left sidebar. You can manage your menus using the drag and drop functionality.
After setting up your menu, select the menu you just created from the theme locations drop-down box and hit the "Save" button.

Scrolling navigation for your one-page:

Add scrolling navigation to your one-page with these easy steps:

  • Step 1 Go to edit your one-page > Enter a Section ID for your outer row elements. Fx. you have a row containing your portfolio items, then click on the edit icon for the outer row and enter "Portfolio" into the Section ID field.
  • Step 2 Next go to Appearance > Menus > Insert a menu item (Note: it's important that you insert your one-page) > Now simply fill in the Section ID with the same ID you just assigned in the page builder.
TIP: To read more about WordPress Menus: WordPress Menu User Guide

6) Hero Section

Return to top

Willow comes with a very customizable hero section. Choose between image slider or video background plus the 3 different text styles. To setup your hero section, follow these steps:

  • Step 1 Go to Hero Slides > Add New > Enter the hero slide option fields.
  • Step 2 Next go to edit your one-page. Check the "Enable Hero Section" option.
  • Step 3 This will bring up a new set of options. Pick the slides you wish to show, choose background overlay and enable/disable parallax effect.

8) Footer

Return to top

The footer can contain widgets, on our demo site we're having contact information and a contact form, this is created with widgets.

You can customize the amount of columns in the footer by going to Theme Options > Footer

To add widgets to your footer, follow these steps:

  • Step 1 Go to Appearance > Widgets > Drag your preferred widgets into the Footer widget areas.
  • Step 2 Use the "Willow: Icon List" widget and follow the instruction in order to get the same widgets as we're using on the demo site.
  • Step 3 To get the contact form showing you first need to have the plugin Contact Form 7 installed.
  • Step 4 Go to Contact > Add New > Enter your information.
  • Step 5 In the "Form" field, make sure to enter this bit of code: See Code
  • Step 6 Save your changes and you will see a shortcode at the top. Copy the shortcode and go back to your Widgets. Now insert a Text Widget and paste the shortcode in.

10) Updating Visual Composer

Return to top

You won't be able to update the Visual Composer plugin on your own. Instead we will send out new theme updates that include the latest version of VC.

When a new Willow update has been released go a head and download Willow again from your downloads page on Themeforest and upload the theme.

Now in order to also update Visual Composer you need to delete the current plugin by going to Plugins > deactivate Visual Composer > Delete Visual Composer.
Once this is done a message will appear on your dashboard telling you to install Visual Composer, follow the instructions and you will have the latest version.
